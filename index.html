¿Cómo contribuir?

  1. Ver si hay documentación/README/wiki de cómo contribuir
  2. Abrir una issue contando lo que echas en falta
  3. Hacer un fork del proyecto
  4. Desarrollar tu proyecto
  5. Pull Request (o como se llame)
  6. Esperar a que la acepten
